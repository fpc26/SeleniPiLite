## [0.1.0] - Initial Release

### Added
- Lunar phase tracking and display
- Support for Waveshare 2.13" e-ink display (touch and non-touch)
- Touch interface with Prev/Next navigation
- File backend for desktop testing
- Location customization (latitude, longitude, timezone)
- Date selection for viewing specific dates
- Auto-clear functionality for e-ink display
- Touch calibration utility
- Raspberry Pi setup scripts
- Environment verification script

### Features
- Real-time lunar phase calculation using Skyfield
- Moon rise/set times
- Illumination percentage
- Phase angle display
- Rotation support (0°, 90°, 180°, 270°)
- Multiple display backend support
- Touch polling and gesture detection
- Power/sleep zone functionality

### Hardware Support
- Raspberry Pi Zero W
- Raspberry Pi Zero 2 W
- Waveshare 2.13" Touch E-Paper HAT
- Waveshare 2.13" E-Paper HAT (non-touch)

### Documentation
- Comprehensive README with setup instructions
- Quick start guide
- Touch troubleshooting guide
- Raspberry Pi Zero setup tips
- Hardware requirements
